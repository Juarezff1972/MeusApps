I have gone to many a links and have found very few actual pages/sites that have the information I need instead of just more links to another page with more links. For this reason, I will start to create this links page of links to pages that have the information I am looking for rather than just more links.
If you have a site or know of a site that is devoted to DOS programming, and has good information on it rather than just a bunch of links, I would like to see it. Please send me an e-mail message with the preferred language in the subject line, and the address and a brief description in the body. I may include it on this page (unless otherwise noted by you).


For all the DOS commands and their parameters, take a look at http://log.on.ca/users/rhwatson/dos7/index.html

Do you have Dissecting DOS by Michael Podanoffsky?
Here is a quote from his page: "Th[is] book contains the full source code to RxDOS. It provides a thorough explanation of the internal architecture and is an excellent reference source." [Api Software]
His book has source code to a full work alike DOS operating system. You can go to his home page and read more about it at http://rxdos.com and http://rxdos.com/dissectingDOS.html.

View http://www.x86.org/ for info on the latest N-TEL secrets/bugs/etc. including undocumented stuff.

Hugi Size Coding Compo. About once a month there is a new assembler size coding compo here. As of this update, it is on the 5th compo. It is great compo'en and is updated 'daily' so that you can see where you are ranked. You can always send in your updated entry.

And more great assembler contests, check out http://www.ctv.es/USERS/guino/contests.htm

Microsoft is a source of information on new products, patches and fixes, upgrades, and news on Microsoft Software. For those few of you that don't know yet.

Doctor Dobb's Journal is a great source of programming information (Windoze). Their site includes a lot of source code that has been included in there journals since early 90's. Subscriptions are at about $25 (US) a year.

Gateway 2000 is the fine company were I bought my machine. Great products and excellent service. Check 'em out.

Browse and Buy Books at amazon.com

The Small Computer Book Club A great source of programming, technical, and informational books. This is where I got most of mine.

Windows95/WinFiles This is were I go for up to date info on all my Windoze 95 based apps. It has upgrades-fixes, tips, shareware, etc.

Simtel Another complete archive of DOS (and Windoze) shareware.

Simtel at CDROM.COM Another SIMTEL mirror (incase one of the others are down).

The Hornet archive- more files.... The Hornet Archive is shutting down soon. See their page for more info.

HelpPC 2.1 A very good archive of information.

Winer.zip A very good archive of information as well as a free book.

Ralf Browns Interrupt lists (Release 60) A very good archive of information about all the interrupts and their services.
For an HTML version of the interrupt list, see http://www.ctyme.com/RBROWN.HTM

Matjaz Trtnik aka maLi's HoMePaGe about 3D graphics programming A friends programming page of 3D programming

What's it's file format If you are looking for a specific file format, this place has just about every file format needed.

EdAlchemy Homepage. Excellent, let me say that again, Excellent graphics. If you want to program games, this is the place to be.
