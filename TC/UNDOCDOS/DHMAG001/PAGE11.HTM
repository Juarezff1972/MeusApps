C


This C column is one of four columns that will be in every issue, with the other three being the other 'major' compilers/assemblers; Assembly, Basic, and Pascal.

As with the other columns to there respected names, this column will be devoted to C and C programming. Each issue will have a new article or a continuing article about the C language.

Unless otherwise noted, I will use the MS-Quick C 2.5x compiler for each of the C listings in this column.

As this is the first issue, I have written an article in the Assembler Column on page 9 for the four languages. Starting with the next issue, this column will have its own article specifically about C and the C language. ¥
