An EMS tester/information viewer

EMSINFO - an EMS memory tester that displays the info of your EMS
Version: 1.00
Author: Ben Lunt (Forever Young Software(R))
Date: 04/14/97
Assembler: MASM 5.10

Download the EMS tester zip file from here, or below, and unzip it (PKUNZIP 2.04g recommended) in a directory of your choice, then view EMSINFO.TXT (To download, I suggest right clicking the link and saving to disk)

EMSINFO.ZIP (4,304) should contain 3 files:
EMSINFO.ASM (11,146)
EMSINFO.EXE (1,687)
EMSINFO.TXT (892)
If there are any differences in the package you got, please let me know.

Type EMSINFO at the DOS prompt and EMSINFO will display the information about your current EMS status.
